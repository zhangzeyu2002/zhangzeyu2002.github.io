# 个人主页设计方案

## 方案一：极简科技主义（Minimalist Techno）
**设计运动**：极简主义 + 当代科技美学

**核心原则**：
- 信息优先：内容排列清晰，避免装饰性元素
- 留白哲学：充足的空间让每个元素呼吸
- 单色系统：深灰/黑色背景 + 亮色强调
- 几何精准：使用网格系统和精确的对齐

**色彩哲学**：
- 主背景：深灰色（#0F1419）
- 文字：浅灰/白色（#E8EAED）
- 强调色：霓虹蓝（#00D9FF）或霓虹绿（#39FF14）
- 情感意图：专业、前沿、科技感强

**布局范式**：
- 非对称网格：左侧固定导航栏，右侧内容流
- 卡片堆叠：项目和经历以卡片形式垂直排列
- 动态分割线：使用渐变或动画分割各个板块

**标志性元素**：
- 代码块风格的文字排版
- 微妙的网格背景纹理
- 悬停时的发光效果

**交互哲学**：
- 流畅的过渡动画
- 鼠标悬停时的微妙发光
- 平滑的滚动效果

**动画指南**：
- 元素进入时的淡入 + 向上滑动（200ms）
- 按钮悬停时的发光脉冲
- 卡片悬停时的轻微提升和阴影增强

**排版系统**：
- 标题：IBM Plex Mono Bold（代码感）
- 正文：Inter Regular（可读性）
- 强调：IBM Plex Mono 用于技术术语

## 方案二：学术优雅主义（Academic Elegance）
**设计运动**：新古典主义 + 学术传统

**核心原则**：
- 学术严谨：模仿学术论文的排版风格
- 经典优雅：衬线字体 + 传统色彩
- 信息层级：清晰的视觉权重区分
- 可读性至上：充足的行距和字体大小

**色彩哲学**：
- 主背景：奶白色（#FAFAF8）
- 文字：深棕色（#2C2C2C）
- 强调色：学术蓝（#1E3A5F）
- 次强调：金色（#D4AF37）
- 情感意图：可信、专业、学术权威

**布局范式**：
- 对称中心布局：主要内容居中，宽度受限
- 分栏设计：左侧简历，右侧研究方向
- 横向分割线：使用细线分隔各个板块

**标志性元素**：
- 学术徽章/勋章图标
- 引用块样式的推荐语
- 脚注式的补充信息

**交互哲学**：
- 保守的交互设计
- 悬停时的下划线或背景色变化
- 平稳的页面转换

**动画指南**：
- 元素进入时的淡入（300ms，较缓）
- 悬停时的下划线展开动画
- 页面加载时的渐进式内容显示

**排版系统**：
- 标题：Playfair Display Bold（优雅衬线）
- 正文：Lora Regular（学术衬线）
- 强调：Lora Italic 用于引用

## 方案三：创意现代主义（Creative Modernism）
**设计运动**：当代艺术 + 数字创意

**核心原则**：
- 视觉冲击：大胆的配色和排版
- 创意表达：非线性布局，打破常规
- 动态感：充满运动和变化
- 个性突出：展现创意和独特性

**色彩哲学**：
- 主背景：深紫色渐变（#1A0033 → #2D0052）
- 文字：亮紫色/粉色（#E0B0FF）
- 强调色：霓虹粉（#FF006E）和霓虹青（#00F5FF）
- 情感意图：创意、年轻、充满能量

**布局范式**：
- 非对称分布：内容块随意排列，但保持视觉平衡
- 浮动卡片：卡片以不同角度和深度浮动
- 曲线分割：使用 SVG 曲线分割各个板块

**标志性元素**：
- 动画背景粒子或渐变
- 旋转的装饰元素
- 3D 变换效果的卡片

**交互哲学**：
- 充满活力的交互反馈
- 悬停时的旋转、缩放或倾斜
- 鼠标跟踪效果

**动画指南**：
- 元素进入时的弹性动画（cubic-bezier）
- 悬停时的 3D 旋转或缩放（1.05x）
- 背景粒子的连续漂浮动画
- 页面加载时的渐进式展开

**排版系统**：
- 标题：Space Grotesk Bold（现代无衬线）
- 正文：Outfit Regular（友好无衬线）
- 强调：Space Grotesk Bold 用于技术术语

## 选定方案：极简科技主义（Minimalist Techno）

**原因**：
作为人工智能硕士生，极简科技主义最能体现你的专业身份。这个设计方案：
- 展现技术素养和专业性
- 简洁高效，不分散注意力
- 强调内容而非装饰
- 与科技行业审美一致
- 易于维护和扩展

**设计决策**：
- 深色主题：符合科技行业审美，减少眼睛疲劳
- 霓虹强调色：现代感，突出重点
- 代码风格排版：体现技术背景
- 流畅动画：提升交互体验，但不过度
